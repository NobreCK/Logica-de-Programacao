# Conversor-USDBRL
Conversor de moedas (USD para BRL) em Python utilizando a AwesomeAPI para cotações em tempo real.

Este é um script em Python que realiza a conversão de Dólar (USD) para Real (BRL) consumindo dados ao vivo de uma API financeira.

##  Tecnologias Utilizadas
* **Python 3**
* **Biblioteca Requests:** Para comunicação com a API.
* **AwesomeAPI:** Fonte dos dados de cotação em tempo real.

##  O que este projeto demonstra:
1. **Consumo de APIs:** Como buscar dados externos via HTTP.
2. **Tratamento de Erros:** Uso de `try/except` para lidar com entradas inválidas ou falhas de conexão.
3. **Manipulação de JSON:** Como extrair informações específicas de dicionários retornados pela web.
